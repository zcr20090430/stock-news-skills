# stock-news-analyzer

股票资讯分析与走势预测技能，整合多源资讯搜索与技术分析，生成综合分析报告。

## 功能概述

- **多源资讯搜索**：同时抓取金十数据、雪球、东方财富、新浪财经、财联社、政策新闻、公司公告、Yahoo Finance、行业新闻等多个数据源
- **股价数据获取**：支持 A股（沪深）、港股、美股，获取近三个月 K 线数据
- **技术分析**：均线系统（MA5/10/20/60）、MACD、KDJ/RSI、量价分析、支撑压力位、斐波那契回撤、波浪理论
- **综合研判**：结合基本面（新闻资讯）与技术面，给出短期/中期走势预测及风险提示

## 触发场景

- 询问某只股票的资讯和走势（如"分析一下茅台的走势"）
- 搜索股票相关新闻并分析影响
- 技术分析请求（支撑位、压力位等）
- 股价走势预测

## 脚本说明

| 脚本 | 功能 |
|------|------|
| `scripts/multi_source_search.py` | 多源资讯搜索 |
| `scripts/stock_data.py` | 股价 K 线数据获取 |
| `scripts/technical_analysis.py` | 技术指标计算与分析 |
| `scripts/fundamental_analysis.py` | 基本面分析 |
| `scripts/jin10_scraper.py` | 金十数据爬虫 |
| `scripts/xueqiu_scraper.py` | 雪球数据爬虫 |

## 输出格式

生成 Markdown 格式分析报告，包含：多源资讯汇总、股价走势分析、技术分析（均线/MACD/KDJ/量价/波浪理论）、综合研判、走势预测及风险提示。
