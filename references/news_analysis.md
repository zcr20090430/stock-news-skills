# 新闻分析框架

## 新闻分类

### 按重要性
1. **重大利好**
   - 政策支持（行业利好、税收优惠等）
   - 重大合同签订
   - 产品技术突破
   - 行业龙头地位确立
   - 大额订单

2. **一般利好**
   - 业绩预增
   - 分红送转
   - 股东增持
   - 机构调研

3. **重大利空**
   - 政策限制
   - 业绩暴雷
   - 法律诉讼
   - 高管变动
   - 产品问题

4. **一般利空**
   - 业绩下滑
   - 股东减持
   - 行业竞争加剧

### 按时效性
1. **突发新闻**：立即影响，短期波动大
2. **周期新闻**：业绩报告、宏观数据，可预期
3. **长期趋势**：行业政策、技术变革，影响深远

## 新闻影响评估

### 影响力度判断
| 因素 | 高影响 | 中影响 | 低影响 |
|------|--------|--------|--------|
| 来源权威性 | 官方发布 | 行业媒体 | 社交平台 |
| 影响范围 | 行业级 | 公司级 | 部门级 |
| 持续时间 | 长期 | 中期 | 短期 |
| 市场预期 | 超预期 | 符合预期 | 不及预期 |

### 时间节点对应
1. **盘前新闻**：影响开盘价
2. **盘中新闻**：实时影响，波动剧烈
3. **盘后新闻**：影响次日开盘
4. **周末/假期新闻**：影响下一个交易日

- **优势**：速度快、来源可靠

### 雪球
- **特点**：用户讨论多，观点多元
- **优势**：情绪感知、草根观点
- **适用**：市场情绪、散户观点

## 分析方法

### 1. 时间线梳理
- 按时间排序所有相关新闻
- 识别关键事件节点
- 对应股价变化

### 2. 情绪分析
- 统计利好/利空新闻数量
- 判断市场情绪倾向
- 对比股价走势验证

### 3. 相关性分析
- 新闻事件 → 股价变动
- 判断因果关系
- 识别市场反应模式

### 4. 预期管理
- 对比市场预期与实际
- 判断是否已被定价
- 发现预期差机会

## 报告撰写要点

### 结构
1. **摘要**：核心结论（2-3句话）
2. **新闻汇总**：时间线形式
3. **股价对应**：新闻+价格走势
4. **分析解读**：因果关系
5. **后续展望**：基于信息预判

### 写作原则
1. 客观陈述事实
2. 区分事实与观点
3. 标注信息来源
4. 注明时间节点
5. 给出判断依据

## 注意事项

### 信息验证
1. 交叉验证多个来源
2. 关注官方公告
3. 警惕谣言和假消息
4. 注意旧闻重提

### 风险提示
1. 新闻滞后性
2. 信息不完整
3. 解读偏差
4. 市场非理性反应

### 禁止事项
1. 不做投资建议
2. 不承诺收益
3. 不预测具体价位
4. 不涉及内幕信息
