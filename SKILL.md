# 股票资讯分析技能

整合多源资讯与技术分析，提供股票走势综合分析报告。

## 工作流程

### 1. 多源资讯搜索

使用综合搜索脚本，同时搜索多个数据源：

```bash
python3 ~/.qclaw/skills/stock-news-analyzer/scripts/multi_source_search.py "<股票名称或关键词>" 90 "<股票代码>"
```

**数据源包括**：
- **金十数据** - 实时快讯，重大市场新闻
- **雪球** - 用户讨论，市场情绪
- **东方财富** - 财经新闻，研报资讯
- **新浪财经** - 综合财经新闻
- **财联社** - 快讯速报
- **政策新闻** - 相关政策法规
- **公司公告** - 官方公告信息
- **行业新闻** - 行业动态趋势

根据这些数据源的信息, 分析当前股票的PE,以及远期盈利能力,和未来面临的同品类公司竞争关系, 以及当前国际实事,政策对公司未来发展的前景展望;

### 2. 获取股价数据

获取最近三个月（约90个交易日）的K线数据：

```bash
python3 ~/.qclaw/skills/stock-news-analyzer/scripts/stock_data.py "<股票名称或代码>" 90
```

支持的市场：
- A股（沪深）
- 港股
- 美股

### 3. 技术分析

对K线数据进行全面技术分析：

```bash
python3 ~/.qclaw/skills/stock-news-analyzer/scripts/technical_analysis.py -d '<klines_json>'
```

**技术分析内容**：

#### 3.1 均线系统
- MA5/MA10/MA20/MA60
- 多头/空头排列判断
- 均线支撑压力

#### 3.2 MACD指标
- DIF/DEA/MACD柱
- 金叉/死叉信号
- 顶底背离识别

#### 3.3 KDJ/RSI指标
- 超买超卖判断
- 金叉死叉趋势

#### 3.4 量价分析
- 放量/缩量上涨下跌
- 量价背离
- 成交量趋势

#### 3.5 支撑位/压力位
- 前期高低点
- 均线位置
- 整数关口
- 成交密集区

#### 3.6 斐波那契回撤
- 23.6%、38.2%、50%、61.8%、78.6%回撤位
- 波段高低点识别
- 关键位置标注

#### 3.7 波浪理论分析
- 波浪识别（1-5浪、A-B-C浪）
- 当前浪型判断
- 波浪比例分析
- 基于浪型的预测

### 4. 综合分析

#### 4.1 时间节点对应
将新闻事件按时间排序，与股价走势对应：
- 识别关键新闻发布日期
- 查看对应日期的股价变化
- 分析因果关系

#### 4.2 量价分析
根据 `references/technical_analysis.md` 中的量价关系表分析：
- 放量上涨/下跌的含义
- 缩量上涨/下跌的含义
- 成交量异常变化

#### 4.3 波浪分析
结合波浪理论判断当前所处阶段：
- 推动浪（1-5浪）：上升趋势
- 调整浪（A-B-C浪）：下降趋势
- 各浪型特征和预测目标

### 5. 预测分析

#### 基本面因素（来自新闻）
- 政策影响
- 行业趋势
- 公司业绩
- 市场情绪

#### 技术面因素
- 均线趋势
- MACD/KDJ/RSI信号
- 成交量变化
- 支撑压力位
- 波浪位置

#### 综合判断
| 情况 | 建议 |
|------|------|
| 基本面+技术面都看多 | 看多 |
| 基本面+技术面都看空 | 看空 |
| 基本面看多，技术面看空 | 震荡偏多，注意技术回调 |
| 基本面看空，技术面看多 | 震荡偏空，注意反弹 |

## 注意事项

1. **数据获取失败处理**
   - 若某个数据源获取失败，继续使用其他来源
   - 可使用 web_fetch 补充关键新闻

2. **波浪理论说明**
   - 波浪识别具有主观性，仅供参考
   - 需结合其他技术指标综合判断
   - 不同时间周期可能有不同浪型

3. **分析客观性**
   - 区分事实陈述与观点判断
   - 同时呈现多空双方观点
   - 注明信息来源

4. **风险提示**
   - 必须包含免责声明
   - 列出主要风险点
   - 不做具体价位预测
